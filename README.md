# 🧠 MeetMind — AI Meeting Intelligence

> Transform meeting recordings and transcripts into summaries, action items, and decisions — instantly.

![AI Powered](https://img.shields.io/badge/AI-Powered-7c6dfa?style=flat-square) ![Free](https://img.shields.io/badge/100%25-Free-6dfacc?style=flat-square) ![No Backend](https://img.shields.io/badge/No%20Backend-Required-fa6d9a?style=flat-square)

## ✨ Features
- 🎙 Live voice recording via Web Speech API (free, no API needed)
- 📝 Paste any meeting transcript
- 🤖 AI-powered analysis via Groq + Llama 3 (free tier)
- Extracts: Summary · Key Decisions · Action Items with owners & deadlines · Follow-up Questions
- One-click copy for action items
- API key saved locally — never leaves your browser

## 🚀 Quick Start
1. Get a free API key at [console.groq.com](https://console.groq.com) (no credit card)
2. Open the live app, paste your key, hit Save
3. Record or paste a transcript → click Analyze

## 🛠 Tech Stack
| Tech | Purpose | Cost |

## 🔒 Privacy
No backend. No database. Your API key stays in your browser's localStorage only.
